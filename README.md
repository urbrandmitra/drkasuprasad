# Dr. Kasu Prasad Reddy — Authority Website

This repository contains the development build of the official archival website for Dr. Kasu Prasad Reddy.

## Purpose
This website is designed as a permanent public record of:
- Clinical contributions
- Research and publications
- Institutional impact
- Professional legacy

It is not a clinical service or appointment platform.

## Tech Stack
- Next.js (App Router)
- TypeScript
- Custom CSS (no UI frameworks)

## Structure
- /src/app → Pages and layout
- /src/app/components → Header, Footer, shared components
- /src/app/styles → Page-level styles
- /public → Images, logos, assets
